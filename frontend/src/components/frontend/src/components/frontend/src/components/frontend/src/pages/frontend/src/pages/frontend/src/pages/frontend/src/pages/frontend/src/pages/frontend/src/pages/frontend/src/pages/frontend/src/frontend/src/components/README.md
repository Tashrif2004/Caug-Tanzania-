# CAUGU Tanzania Marketplace

A full-stack business directory and marketplace platform for Tanzania.

## 🚀 Features

- User authentication (Register/Login)
- Business listings with search and filters
- Business reviews and ratings
- Subscription plans (Free, Featured, Premium)
- Verified business badges
- User dashboard
- Responsive design

## 🛠️ Tech Stack

### Backend
- Node.js with Express.js
- MongoDB with Mongoose
- JWT Authentication
- Bcrypt for password hashing

### Frontend
- React.js with Hooks
- React Router for navigation
- Axios for API calls
- React Icons for beautiful icons
