## 🌍 About CAUGU

**CAUGU** is a Tanzanian business directory and marketplace platform that connects customers with verified service providers. Find electricians, plumbers, photographers, doctors, and more in seconds!

### ✨ Features

- 🔐 **User Authentication** - Register and login securely
- 🔍 **Smart Search** - Find businesses by name, category, or location
- ⭐ **Reviews & Ratings** - Rate businesses and read customer feedback
- 💼 **Business Dashboard** - Manage your business listing
- 🏷️ **Subscription Plans** - Free, Featured, and Premium options
- ✅ **Verified Badges** - Trusted businesses get verified status
- 📱 **Mobile Responsive** - Works on all devices

## 🛠️ Tech Stack

### Backend
- **Node.js** with Express.js
- **MongoDB** with Mongoose ODM
- **JWT** for authentication
- **bcrypt** for password hashing

### Frontend
- **React.js** with Hooks
- **React Router** for navigation
- **Axios** for API calls
- **React Icons** for beautiful icons
